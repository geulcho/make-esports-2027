# Intermatch & WE Qualifiers Schedule PRD

**Document name:** Intermatch & WE Qualifiers PRD  
**Version:** v1.0  
**Scope:** W7~8, W25~26, W31~32 national-team windows  
**WE main event:** Pending, to be defined separately  
**Language:** Korean

---

## 1. Purpose

이 문서는 국가대표 인터매치 기간에 진행되는 **WE(World Event) 예선**, **IQ(Interregional Qualifier)**, 그리고 W31~32 기간의 보조 국가대표 대회인 **EEC**와 **TPC**의 일정을 정리한다.

전체 구조는 다음과 같다.

| 기간 | 역할 |
|---|---|
| **W7~8** | 권역별 예선 전반부 |
| **W25~26** | 권역별 예선 후반부 + 권역별 플레이오프 + 중동아프리카 최종 예선 |
| **W31~32** | IQ, EEC, TPC, 친선전 슬롯 |
| **W45~48** | WE 본선, 추후 별도 정의 |

---

## 2. Final WE Qualification Summary

WE 본선에는 총 **24개 국가대표 팀**이 진출한다.

| 경로 | 팀 수 |
|---|---:|
| 유럽 직행 | 8 |
| 아시아태평양 직행 | 6 |
| 아메리카 직행 | 4 |
| 중동아프리카 직행 | 1 |
| IQ 와일드카드 | 5 |
| **합계** | **24** |

# 4. Regional Qualifier Formats

| 항목 | 내용 |
|---|---:|
| 참가국 | 32개국 |

# 8. Friendly Match Policy for W31~32

W31~32에는 이미 WE 진출이 확정된 국가들이 다음 목적의 친선전을 진행할 수 있다.

| 목적 | 설명 |
|---|---|
| 실전 합 맞추기 | WE 본선 전 국가대표 로스터 점검 |
| 권역 외 상대 적응 | 서로 다른 메타와 스타일 경험 |
| 방송 콘텐츠 | 지역별 국가대표 콘텐츠 확보 |
| 예비 일정 활용 | IQ/EEC/TPC가 없는 날짜의 보조 슬롯 사용 |

권장 친선전 슬롯은 다음과 같다.

| 날짜 | 권장 활용 |
|---|---|
| W31 월 | IQ 추첨 후 저녁 친선전 가능 |
| W31 수 | EEC 외 타 권역 친선전 가능 |
| W31 금 | TPC 외 유럽/MEAF 친선전 가능 |
| W31 일 | TPC 외 유럽/MEAF 친선전 가능 |
| W32 월 | 가장 강력한 친선전 권장일, 전 대회 예비일 겸용 |
| W32 수 | TPC 외 유럽/MEAF 친선전 가능 |
| W32 금 | EEC 외 APAC/Americas/MEAF 친선전 가능 |

친선전은 WE 본선 진출권에 영향을 주지 않는다.
