# L_TR 8팀 리그 진행 방식 PRD

**문서 버전:** v1.0  
**대상 리그:** 터키 `L_TR`  
**리그 구조:** 8팀 단일 경기장 리그  
**리그 체제:** 스프링 / 서머 2스플릿  
**정규시즌 경기 방식:** Bo1  
**플레이오프 경기 방식:** 승자조 1라운드 Bo3, 그 외 전 경기 Bo5  
**연간 우승팀:** 스프링 우승팀 1팀, 서머 우승팀 1팀

> 해석 기준: 사용자가 말한 “더블라운드 로빈”은 정규시즌 포맷으로 반영한다. 플레이오프는 `L_WEU` / `L_NEU`와 동일한 **8팀 더블 엘리미네이션** 구조로 반영한다.

---

## 1. 목적

이 PRD는 터키 `L_TR` 리그가 52주 글로벌 캘린더 안에서 스프링과 서머를 운영하기 위한 기준 문서다.

핵심 목표는 다음과 같다.

| 목표 | 설명 |
|---|---|
| 8팀 완전 참가형 리그 | 정규시즌은 시드 결정, 플레이오프는 8팀 전원 진출 |
| 단일 경기장 운영 | 모든 경기는 하나의 경기장에서 진행 |
| 주말 정규시즌 | 정규시즌은 토요일/일요일만 사용 |
| 짧고 선명한 정규시즌 | 7주 동안 Bo1 더블 라운드 로빈 완료 |
| L_WEU/L_NEU식 포스트시즌 | 승자조 1라운드만 Bo3, 나머지는 Bo5 더블 엘리미네이션 |

## 3. 정규시즌 구조

정규시즌은 8팀 더블 라운드 로빈으로 진행한다.

| 항목 | 수치 |
|---|---:|
| 팀 수 | 8팀 |
| 팀당 경기 수 | 14경기 |
| 전체 경기 수 | 56경기 |
| 총 매치데이 | 14일 |
| 매치데이당 경기 수 | 4경기 |
| 정규시즌 기간 | 7주 |
| 주간 경기일 | 토요일 / 일요일 |
| 주당 경기 수 | 8경기 |

계산식은 다음과 같다.

```text
8팀 더블 라운드 로빈 = 8팀 × 14경기 ÷ 2 = 56경기
7주 × 주 2일 × 하루 4경기 = 56경기
```

각 매치데이에는 8팀 모두가 정확히 한 번씩 경기한다.
